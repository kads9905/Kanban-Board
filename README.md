# Kanban Board

A lightweight Kanban Board built with vanilla JavaScript for managing tasks through different stages of completion.

## Features

* Create and delete tasks
* Drag and drop tasks across columns
* Real-time task count updates
* Persistent storage using Local Storage
* Clean and responsive dark-themed interface

## Built With

* HTML5
* CSS3
* JavaScript (ES6)

## Getting Started

Clone the repository and open `index.html` in your browser.

```bash
git clone <repository-url>
cd kanban-board
```

No additional setup or dependencies are required.

## Project Structure

```text
kanban-board/

```

## Preview

The board allows users to organize tasks into **To Do**, **In Progress**, and **Done** columns while supporting drag-and-drop interactions and automatic data persistence.
